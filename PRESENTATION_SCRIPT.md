### **[15-60 SECONDS] - MAIN PROJECT: SPORTNOVA** ⭐⭐⭐
```
"This is SportNova – an AI-powered sports analytics platform I built with a team of peers 
to help athletes track and optimize their performance in real-time.

Here's what it does: Athletes can upload training videos and performance metrics, 
and the platform uses Google Gemini AI to analyze them and give personalized training insights.

I led the full-stack development:
• Built the React dashboard with interactive charts and smooth animations
• Designed REST APIs in Node.js and Express for the backend
• Set up MongoDB for data storage
• Implemented JWT-based authentication for role-based access control

The biggest technical challenge was handling large video uploads without freezing the UI. 
I solved this using Multer for streaming uploads, offloaded heavy processing to background jobs, 
and added real-time progress indicators on the frontend."
```
